Technical Capabilities

Architecture designed for scale
01

Large World Scale

Supports terrain maps exceeding 100 km² with streaming geometry, LOD systems, and background world simulation — designed for the scale Arma demands.

02

Cross-Platform Architecture

Native builds for PC, Xbox Series X|S, and PlayStation 5. Shared codebase with platform-specific performance optimisation layers for each target.

03

EnScript Scripting API

A full-featured, type-safe scripting language modders use to create everything from new weapons to entirely new game modes. Identical to what BI devs use internally.

04

Networked Simulation

Server-authoritative multiplayer with client-side prediction and a custom replication layer supporting 128+ simultaneous players in a shared physics simulation.
